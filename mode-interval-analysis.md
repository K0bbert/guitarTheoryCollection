## Interval Legend
- **R** = Root / Unison
- **m2** = Minor 2nd (half step)
- **M2** = Major 2nd (whole step)
- **m3** = Minor 3rd
- **M3** = Major 3rd
- **P4** = Perfect 4th
- **TT** = Tritone (Augmented 4th / Diminished 5th)
- **d5** = Diminished 5th
- **P5** = Perfect 5th
- **m6** = Minor 6th
- **M6** = Major 6th
- **m7** = Minor 7th
- **M7** = Major 7th

## Notes
- Each scale degree shows the intervals available from that note to other notes within the same mode
- This analysis is useful for understanding chord construction and harmonic possibilities within each mode
- The intervals listed represent the available harmonic material when building chords or melodies from each scale degree
