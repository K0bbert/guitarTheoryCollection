# Key Harmony Map

Use this as a quick songwriting overview for the selected key in the sidebar Root Note selector.

- Middle row: diatonic major-key triads I to vi, following M m m M M m.
- Top row: matching secondary dominants with the suggested resolution chord.
- Bottom row: modal interchange options borrowed from the parallel minor for I, IV, ii, and V functions.

Bottom row:
iv and ii° are borrowed as standalone color/texture chords — they create a melancholic or dramatic feel without cleanly substituting for any single diatonic function. bIII, bVI, and bVII each share strong common tones with a specific diatonic chord, making them clear functional substitutes. iv and ii° are pre-dominant in flavor but serve more as passing borrowed chords, so no single "borrow for" target applies.

## Aeolian Map (Natural Minor)

This second map uses common minor-key techniques instead of a simple index rotation from major:

- Aeolian diatonic center: i, ii°, bIII, iv, v, bVI, bVII.
- Applied dominants: commonly used to tonicize bIII, iv, v, bVI, and bVII.
- Minor modal interchange: common color borrowings include I (Picardy), IV, ii, and V.

## Harmonic Minor Map

This third map focuses on harmonic minor behavior (raised 7th) and common real-world color/replacement choices.

- Harmonic minor center: i, ii°, bIII+, iv, V, bVI, vii°.
